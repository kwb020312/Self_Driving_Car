# 자율주행 자동차

1. 자동차를 학습시키기 위하여, 실행된 이후에 사고가 나지 않는 시점 우측의 저장하기(💾)를 눌러 `가장 최적의 Neural`을 저장한다

2. 새로고침 시, 이전에 저장된 `최적의 Neural`을 포함한 999대의 자동차가 추가로 배치되며, 장애물을 모두 통과할 때까지 학습한다.

3. 모든 장애물이 통과되는 경우를 발견하였다면 학습을 종료한다.
